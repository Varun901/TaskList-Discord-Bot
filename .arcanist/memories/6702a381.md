Tests for this bot are placed in `src/test_fixes.py` and configured via `pytest.ini` at the repo root with `asyncio_mode = auto` (from the `pytest-asyncio` plugin). Test fixtures use `tmp_path` and `monkeypatch` to provide each test with an isolated SQLite database by patching both `database.DATABASE_PATH` and `config.DATABASE_PATH`. Environment variables required by `config.py` on import (`DISCORD_BOT_TOKEN`, `TIMEZONE`, `DAILY_POST_HOUR`, `DAILY_POST_MINUTE`, `DATABASE_PATH`) must be set via `os.environ.setdefault()` before any bot modules are imported in the test file.
